## Profile User Api

### `GET` Profile User
```
/api/users/profile
```
Get profile user
#### Request Headers
| Key | Value |
|---|---|
|Accept|application\json
|Authorization|{token_type} {access_token}|

##### Example
| URL | Description |
|---|---|
| /api/users/profile | Get profile of user |
